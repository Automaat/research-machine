# NotebookLM Research Paper Infographic Prompt

Generate an infographic that covers:

## Visual Hierarchy

- Bold headline: main finding/thesis
- 3-5 key insights as primary sections
- Supporting data/examples as subsections
- Clear visual flow (top-to-bottom or left-to-right)

## Data Visualization

- Key statistics with context
- Before/after comparisons
- Process flows or timelines
- Quantitative improvements/results
- Visual metaphors for abstract concepts

## Content Focus

- Problem statement (why this matters)
- Core innovation/approach
- Main results with numbers
- Real-world impact
- Next steps or implications

## Design Principles

- One main idea per section
- Icons/symbols for concepts
- Color coding for categories
- Minimal text, maximum clarity
- Hierarchical sizing (important = bigger)

## Avoid

- Dense paragraphs
- Complex methodology details
- Every data point from paper
- Technical jargon without explanation
- More than 7 sections

Create visual that conveys paper's essence in 30 seconds of viewing.

---

## Usage

Paste as instructions when uploading paper to NotebookLM before generating infographic.

## Customizations

- Add `"Focus on [specific finding]"` for targeted visual
- Add `"Compare [A] vs [B]"` for comparative infographic
- Add `"Style: [minimal/detailed/technical]"` for design approach
- Add `"Audience: [general/technical/executive]"` for complexity level

## Structure Suggestions

**Flow type:** Problem → Solution → Results → Impact

**Comparison type:** Traditional vs Novel Approach (side-by-side)

**Process type:** Step 1 → Step 2 → Step 3 → Outcome

**Impact type:** Key Metric Before → After (with methodology in between)
